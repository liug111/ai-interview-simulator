岗位名称：大数据开发工程师（实习/应届）

岗位职责：
1. 参与大数据平台的搭建与维护
2. 负责数据采集、清洗、存储和分析处理
3. 使用 Hadoop/Spark 等技术进行数据处理任务开发
4. 参与数据可视化方案的设计与实现
5. 配合团队完成数据产品的开发和优化

任职要求：
1. 计算机、大数据相关专业本科及以上学历
2. 熟悉 Hadoop 生态（HDFS、MapReduce、Hive、Spark）
3. 掌握 Python 或 Java，有良好的编程习惯
4. 了解 Linux 基本操作和 Shell 编程
5. 了解数据仓库和 ETL 流程
6. 有大数据相关项目或实习经验者优先
